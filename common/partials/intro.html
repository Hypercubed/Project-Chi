Visualization website toolkit

Project χ is a modular open-source visualization gallery toolkit. It offers a boilerplate for building self-hosted data-centric visualization websites. Geared towards sharing of supplemental materials associated with scientific publications; Project χ allows visitors to interact with visualizations, download associated data and images, and even try the visualization with their own datasets. For developers the framework comes preconfigured with the popular design frameworks (Twitter Bootstrap, Angular UI), cutting edge build tools (gulp, jspm, SystemJS), and tools necessary for quickly integrating interactive visualizations using d3.js and AngularJS.
